# Interactive Rebasing for Clean Commit History

## Objective

- Use interactive rebase to tidy up your commit history

## Tasks Performed

### 1. Creating Commits

- Made a series of commits, including minor changes or commits with typos in messages

### 2. Running Interactive Rebase

- Executed the command:
  ```bash
  git rebase -i HEAD~n
  ```
